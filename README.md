# Aviation RUL

Explainable AI for Remaining Useful Life (RUL) prediction in turbofan engines using the C-MAPSS dataset.

## Local package

This repository provides a local Python package named `xai_aviation_rul`.
The package lives under `src/xai_aviation_rul` and can be imported after
installing the project in editable mode:

```bash
python -m pip install -e .
```

Usage example:

```python
from xai_aviation_rul import data_loader, preprocessor, visualizer

# call functions from the submodules
```
